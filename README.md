# ATM Interface - Python

This is a simple command-line ATM interface written in Python.  
It allows users to:

- Check their account balance
- Deposit money
- Withdraw money
- Exit the program

## Features

- Maintains a running balance (starting at ₹1000)
- Handles invalid inputs and insufficient funds
- Simple text-based menu interface

## How to Run

1. Make sure you have Python installed on your computer (Python 3.x recommended).

2. Clone this repository or download the `atm.py` file.

3. Open a terminal or command prompt and navigate to the folder containing `atm.py`.

4. Run the program with the command:

   ```bash
   python atm.py
